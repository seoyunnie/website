# Website

This is my personal website. Its technical primary purpose is to be something like a portfolio website, but it serves
more like a way for me to learn and experiment with React libraries, and even other web frameworks.

This website currently uses [Vite](https://vite.dev/) and [React](https://react.dev/) as its base and
[TanStack Router](https://tanstack.com/router/latest) for page routing, with [Mantine](https://mantine.dev/) for UI.
